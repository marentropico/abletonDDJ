# Atalhos do Ableton Live 12 (Windows)

O Ableton Live 12 possui um dos fluxos de trabalho mais rápidos do mercado graças aos seus inúmeros atalhos de teclado. A versão 12 introduziu várias atualizações de acessibilidade (permitindo navegar em toda a interface pelo teclado), momentary latching e a capacidade de usar atalhos de uma tecla junto com o Shift se o Teclado MIDI do PC estiver ativado.

## 🖥️ 1. Exibição da Interface (Views)
- **Tab**: Alternar entre Session View e Arrangement View.
- **Shift + Tab**: Alternar entre Visão de Dispositivo (Device View) e Visão de Clipe (Clip View).
- **F11**: Entrar ou sair da Tela Cheia (Fullscreen).
- **F12 / F2**: Mostrar/Ocultar a janela do navegador.
- **Ctrl + Alt + W**: Abrir a Segunda Janela.
- **Ctrl + Alt + B**: Mostrar/Ocultar o Navegador (Browser).
- **Ctrl + Alt + I**: Mostrar/Ocultar a Visão de Ajuda/Info (Info View).
- **Ctrl + Alt + L**: Mostrar/Ocultar a Visão de Detalhes (Detail View).
- **Ctrl + Alt + M**: Mostrar/Ocultar o Mixer.
- **Ctrl + Alt + O**: Mostrar/Ocultar a Visão Geral (Overview).
- **Ctrl + Alt + R**: Mostrar/Ocultar Pistas de Retorno (Return Tracks).
- **Ctrl + Alt + S**: Mostrar/Ocultar Canais de Envio (Sends).
- **Ctrl + Alt + P**: Mostrar/Ocultar Janelas de Plug-ins.
- **Ctrl + Alt + G**: Mostrar/Ocultar a Groove Pool.
- **Ctrl + Alt + V**: Mostrar/Ocultar Janela de Vídeo.
- **? (Interrogação)**: Ligar/Desligar aba lateral de informações.

## 🚀 2. Novos Atalhos e Acessibilidade
- **O**: Ligar/Desligar o Metrônomo.
- **Ctrl + Alt + Shift + F**: Congelar / Descongelar Pista (Freeze/Unfreeze).
- **Esc (na interface principal)**: Pular para a barra de título da pista destacada.
- **F6**: Pular sequencialmente entre as principais seções da interface.
- **Tab / Shift + Tab (Modo de Navegação)**: Mover o foco iterativamente sobre botões ou seções.
- **Shift + Letra de Atalho**: No Live 12, se o "Computer MIDI Keyboard" (Tecla M) estiver ativado, pode usar os atalhos de letra única (como A, S, Z) adicionando a tecla Shift.

## 📁 3. Gestão de Projetos e Preferências
- **Ctrl + N**: Novo Live Set.
- **Ctrl + O**: Abrir Live Set.
- **Ctrl + S**: Salvar Live Set.
- **Ctrl + Shift + S**: Salvar Live Set Como...
- **Ctrl + E**: Dividir (Split).
- **Ctrl + Shift + R**: Exportar Áudio/Vídeo.
- **Ctrl + Shift + M**: Exportar/Importar Arquivo MIDI.
- **Ctrl + ,**: Abrir as Preferências do Ableton.
- **Ctrl + Q**: Fechar o Ableton Live.

## ⏯️ 4. Transporte e Reprodução (Transport)
- **Espaço**: Iniciar/Parar reprodução (Play/Stop).
- **Shift + Espaço**: Retomar reprodução a partir do ponto de pausa.
- **F9**: Gravação (Record).
- **F10**: Voltar ao ponto de início do Arranjo.
- **Shift + F9**: Ativar a Gravação Global do Arranjo.
- **O**: Ativar Metrônomo.
- **Ctrl + Shift + F**: Ativar rolagem automática da tela (Follow).
- **NumPad + / NumPad -**: Acelerar / Desacelerar levemente o tempo da música.

## ✂️ 5. Edição Geral (General Editing)
- **Ctrl + Z**: Desfazer (Undo).
- **Ctrl + Y**: Refazer (Redo).
- **Ctrl + C / X / V / D / A**: Copiar, Recortar, Colar, Duplicar, Selecionar Tudo.
- **Delete / Backspace**: Apagar.
- **Ctrl + R**: Renomear selecionado.
- **Ctrl + J**: Consolidar clipes selecionados (Consolidate).
- **Ctrl + E**: Cortar/Dividir o clipe no cursor (Split).
- **Ctrl + I**: Inserir silêncio no Arranjo.
- **Ctrl + Shift + Delete**: Excluir o tempo no Arranjo.
- **Ctrl + Shift + D**: Duplicar o trecho de tempo selecionado.
- **Ctrl + Alt + V**: Colar tempo.
- **Ctrl + Alt + F**: Criar Fade in / Fade out automático nas bordas do clipe.

## 🎚️ 6. Pistas, Mixer e Roteamento
- **Ctrl + T / Shift + T / Alt + T**: Inserir nova pista de Áudio / MIDI / Retorno.
- **Ctrl + G / Shift + G**: Agrupar pistas / Desagrupar pistas.
- **C**: Armar Pista para gravação.
- **S**: Ativar/Desativar Solo da pista.
- **M**: Mutar/Desmutar pista.
- **Ctrl + Clique no Solo / Arm**: Ativar múltiplos Solos/Arms.
- **Enter (no nome da pista)**: Ocultar/Expandir a altura da pista no Arranjo.

## 🎹 7. Edição MIDI, Piano Roll e Teclado
- **M**: Ativar/Desativar teclado do computador como teclado MIDI.
- **B**: Alternar Modo Desenho (Ferramenta Lápis).
- **Ctrl + Shift + M**: Inserir clipe MIDI em branco na área selecionada.
- **Setas**: Mover nota MIDI.
- **Shift + Seta Cima/Baixo**: Mover nota uma oitava.
- **Shift + Seta Esquerda/Direita**: Expandir/reduzir tamanho da nota.
- **Alt + Clique (Velocity)**: Resetar Velocity.
- **Shift + Arrastar**: Mudar Velocity.
- **0**: Desativar/Ativar nota MIDI selecionada.
- **Ctrl + Espaço**: Pre-escutar nota.

## 🔍 8. Navegador e Busca (Browser)
- **Ctrl + F**: Clicar na barra de busca do Browser.
- **Setas**: Navegar pelas pastas ou arquivos / Expandir/Recolher.
- **Enter**: Carregar o arquivo.
- **Shift + Enter**: Pre-escutar (Preview) arquivo no Browser.
- **1 a 7**: Atribuir cor (Tag).
- **0**: Limpar cor.
- **Q**: Ativar Hot-Swap.

## 📏 9. Grade, Quantização e Marcadores
- **Ctrl + U**: Quantizar notas ou áudio.
- **Ctrl + Shift + U**: Abrir Configurações de Quantização.
- **Ctrl + 1 / 2**: Estreitar / Alargar a grade.
- **Ctrl + 3**: Ligar/Desligar Tercinas.
- **Ctrl + 4**: Ativar/Desativar Snap to Grid.
- **Ctrl + 5**: Grade Fixa ou Grade Adaptativa.
- **Alt + Arrastar**: Ignorar grade.
- **Ctrl + L**: Ligar/Desligar área de Loop.
- **Setas Esquerda/Direita (no clipe)**: Mover o "brace" (marcador de Loop).
- **Seta Cima/Baixo (tamanho do clipe)**: Dobrar (x2) ou cortar pela metade (/2) o tamanho do Loop.

## 📉 10. Automação e Fades
- **A**: Mostrar/Ocultar linhas de Automação.
- **F**: Mostrar/Ocultar Fades.
- **Backspace / Delete**: Excluir ponto de automação.
- **Alt + Clique no ponto**: Curvar linha de automação.

## ✂️ 11. Comping
- **Shift + Alt + T**: Inserir Take Lane.
- **T**: Mostrar/Ocultar Take Lanes.
- **Seta Acima/Abaixo**: Selecionar take.
- **Enter**: Promover take.

## 🎛️ 12. Ajuste de Valores e Mapeamento
- **Ctrl + K / M**: Key Map Mode / MIDI Map Mode.
- **Shift + Arrastar**: Ajuste fino do valor.
- **Delete / Backspace (em parâmetro)**: Retornar ao padrão de fábrica.

## 👁️ 13. Zoom e Controle de Tela
- **+ / -**: Zoom In / Zoom Out.
- **Z / X**: Zoom in área de tempo / Zoom out visualização anterior.
- **W**: Ajustar largura do projeto na tela inteira.
- **H**: Ajustar altura de todas as pistas na tela.
- **Shift + Roda do Mouse**: Rolar horizontalmente.
- **Ctrl + Roda do Mouse**: Zoom In/Out.
- **Alt + Roda do Mouse**: Ajustar altura das pistas sob o mouse.
